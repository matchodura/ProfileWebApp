Online-Refrigerator

Online-Refrigerator is a web app developped in ASP.NET Core in MVC pattern that allows user to find recipes based on provided ingredients. It is an app to help with making up an idea for dinner or other meal. It provides adding and viewing items in ingredients and recipes base, with side functions like ranking or calorie calculator.

Technologies/Patterns used
  • APS.NET MVC
  • C#, HTML, JS, CSS
  • .NET Core 3.1
  • Entity Framework Core 6
  • MS SQL
Main Functions
  • Responsive Web App with CRUD operations
  • AJAX calls for searching and sorting to improve user experience
  • Adding ingredients and recipes
  • Finding recipes based on provided ingredients
  • Calculating nutritional values of ingredients
  • Rating recipes by users

BodyMonitorApp

BodyMonitorApp is a Windows desktop app created with WPF in MVVM pattern that allows for users to record their body progress. It's a tool created with users that are trying to lose weight or gain muscle mass in mind. App works fully offline, as data is stored in local database, allowing to see some charts even with internet off!

Technologies/Patterns used
  • C#
  • .NET Framework
  • WPF
  • MS SQL
  • ADO.NET
  • MVVM
Main Functions
  • Creating and updating user account with full validation
  • Adding and editing values of body parts like chest, biceps, hips and more
  • Displaying records on chart in timespans : year, month, week
  • Catalogue of exercices for selected body parts with youtube videos

Messier Object Identification

Messier Object Identification program was a part of my Master's Degree thesis. It's main purpose is recognizing provided photos of space objects a.k.a Messier Objects, and providing propability of correct guess in percents(0-100%). Classifier used in application is Convolutional Neural Network created as Keras model with Tensorflow. Besides that, program provides some image processing tools and full Messier catalogue.

Technologies used
  • Python
  • Tensorflow
  • Keras
  • PyQt
  • Scipy, Numpy
Main Functions
  • Finding propability of Messier Objects based on provided image
  • Editing and creating new Convolutional Neural Network Model
  • Image processing operations - converting to grayscale, resizing, noising
  • Viewing full catalogue of Messier Objects with trivia about them
